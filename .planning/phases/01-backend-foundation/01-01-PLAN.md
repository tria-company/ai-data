The user must execute the following SQL in their self-hosted Supabase dashboard (SQL Editor) or via psql:

```sql
CREATE TABLE IF NOT EXISTS projetos (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  nome TEXT NOT NULL,
  criado_em TIMESTAMPTZ DEFAULT now() NOT NULL
);

-- Enable RLS but allow all operations (single-user app)
ALTER TABLE projetos ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow all operations on projetos" ON projetos FOR ALL USING (true) WITH CHECK (true);
```

This creates the projetos table with:
- id: UUID primary key with auto-generation
- nome: required text field for the project name
- criado_em: timestamp with timezone, defaults to now()
